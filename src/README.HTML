This is a 3D Viewer written in C

To compile, use make all and run the binary ./s21_3D_viewer/3D_viewer

  • This program is used to view .obj files using GTK4 library.
  • It has numerous options to change rendered image.
  • Scaling, rotation, translation, making .jpeg, .bmp, .gif and more.
  • Check makefile for a detailed review.
